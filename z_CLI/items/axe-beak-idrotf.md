# Axe Beak
*Mount*  

- **Cost**: 50 gp

An [axe beak](z_CLI/bestiary/monstrosity/axe-beak-xmm.md)'s splayed toes allow it to run across snow, and it can carry as much weight as a mule. A domesticated axe beak can be purchased in Ten-Towns for 50 gp.

*Source: Icewind Dale: Rime of the Frostmaiden p. 20*
